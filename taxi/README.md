## Прогнозирование заказов такси

### Данные
В распоряжении исторические данные о заказах такси в аэропортах. 

### Задача
Спрогнозировать количество заказов такси на следующий час.

### Используемые библиотеки
- *pandas*
- *sklearn*
- *sweetviz*
- *numpy*
- *matplotlib*
- *statsmodels*
- *lightgbm*
- *catboost*
