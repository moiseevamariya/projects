## Определение негативных и позитивных комментариев при помощи BERT

### Данные
В распоряжении набор данных с разметкой о токсичности правок. 

### Задача
Обучить модель классифицировать комментарии на позитивные и негативные.

- *pandas*
- *sklearn*
- *numpy*
- *lightgbm*
- *catboost*
- *pytorch*
